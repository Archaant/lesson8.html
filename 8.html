Color in HTML

You can set the background color for HTML elements.
HTML supports 140 standard color names.
You can give the text, background and borders any color.
In the following paragraphs you can see examples with their code.

Text color

Text in color red and the font-size is 28px.

<p style="color: red; font-size: 28px;"> Text in color red and the font-size is 28px. </p>


Text background-color

Text background-color is red and the font-size is 28px.

<p style="background-color: red; font-size: 28px;"> Text background-color is red and the font-size is 28px. </p>


Text border color

The text border color is red and the font-size is 28px.

<p style="border: 4px solid red; font-size: 28px;"> The text border color is red and the font-size is 28px. </p>
